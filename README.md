# System Monitoring Script

## 📌 Project Overview
This project is a simple Linux system monitoring script built using Bash.  
It monitors disk usage, memory usage, and displays top resource-consuming processes.

---

## ⚙️ Features
- Disk usage monitoring
- Memory usage monitoring
- Alerts when thresholds are exceeded
- Top CPU consuming processes
- Top Memory consuming processes
- Logging alerts into a file

---

## 🚨 Thresholds
- Disk Usage Alert: 80%
- Memory Usage Alert: 80%

(You can modify values inside script)

---

## ▶️ How to Run

```bash
chmod +x system_monitor.sh
./system_monitor.sh
